⚠️Risk Matrix

R1
Observer SPOF
Single observer is a SPOF. Deploy multiple observers (3× HA) across DC/DR/EXT; ensure systemd auto-restart.
HIGH
R2
Network Partition
Split-brain risk if observer cannot reach both sites. Place master observer on third network (EXT) — independent path.
HIGH
R3
Reinstate Failure
Old primary may fail to reinstate if Flashback not enabled. Enable Flashback Database on both PRIM and STBY; size FRA appropriately.
MED
R4
Lag Limit Exceeded
Apply lag > LagLimit disables FSFO auto-failover. Monitor lag continuously; alert at 50% threshold (15 s).
MED
R5
Non-replayable Ops
ALTER SESSION, UTL_HTTP, UTL_SMTP, DDL inside TX break TAC replay. Code review required; move external calls outside transactions.
HIGH
R6
Old JDBC Drivers
Pre-19c JDBC drivers cannot use TAC. Upgrade all clients to 19c+ ojdbc11.jar; verify in V$SESSION_CONNECT_INFO.
MED
R7
FAN Port Blocked
Firewall blocking ONS port 6200 prevents FAN events delivery. Open cross-site (DC↔DR) + app→RAC before go-live.
MED
R8
Performance Impact
Commit outcome tracking adds ~3% overhead (LTXID table writes). Benchmark before production; verify SLA margins.
LOW
